# 🚀 Student Dashboard Application

This is a full-stack web application built for my internship project. 

## Features
* **Python Flask Backend:** Acts as the server and handles data routing.
* **Responsive HTML/CSS:** The dashboard automatically adjusts for mobile and desktop screens.
* **To-Do List:** Users can add and delete tasks (Communicates directly with Python).
* **Rich Text Notes:** Integrated with Quill.js for text formatting. Notes auto-save to local storage.
* **JavaScript Alarm Clock:** Users can set time-based reminders that trigger an audio alert and visual pop-up.

## How to Run Locally
1. Ensure Python is installed.
2. Install Flask: `pip install flask`
3. Run the server: `python app.py`
4. Open `http://127.0.0.1:5000` in your web browser.
